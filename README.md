## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/PromiseMachio/portfolio.git
```

### Navigate to the project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
